# Nahue dotfiles

This are my dotfiles including bspwm, i3, ligthdm using the lightdm mini greeter, shxkd and termite mad compatible with GNU Stow for simplicity.

## Details

- **OS**: Arch Linux
- **Terminal**: Termite
- **Shell**: Bash
- **WM**: bspwm
- **Theme**: Nordic
- **Fonts**: San Francisco Pro & San Francisco Mono
